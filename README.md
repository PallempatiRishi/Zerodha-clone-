## Getting Started

### 1. Clone the repository

```sh
git clone https://github.com/PallempatiRishi/Zerodha-clone-.git
cd Zerodha
```

### 2. Install dependencies

#### Backend
```sh
cd backend
npm install
```

#### Dashboard (React App)
```sh
cd ../dashboard
npm install
```

#### Frontend (Landing Page)
```sh
cd ../frontend
npm install
```

### 3. Run the apps

#### Start Backend
```sh
cd ../backend
npm start
```

#### Start Dashboard
```sh
cd ../dashboard
npm start
```

#### Start Frontend (Landing Page)
```sh
cd ../frontend
npm start
```
